# 🌦️ Weather API Data Collector (Python)

## 📌 Overview

This project is a Python-based weather data collector that integrates with the OpenWeatherMap API to retrieve real-time weather data for any city worldwide.

The project demonstrates practical API integration and data processing using Python, simulating a simple real-world data collection workflow.

---

## 🛠️ Technologies Used

- Python  
- Requests  
- python-dotenv  
- REST API (OpenWeatherMap)  
- JSON data parsing  
- Datetime handling  

---

## 🚀 Features

- Fetches real-time weather data by city name
- Converts temperature directly to Celsius
- Displays:
  - Current temperature
  - Feels-like temperature
  - Min/Max temperature
  - Humidity
  - Pressure
  - Wind speed & direction
  - Cloudiness
  - Sunrise & Sunset time
  - Last updated timestamp
- Handles API errors gracefully
- Uses environment variables to securely store API keys
